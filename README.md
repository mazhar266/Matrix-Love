# Matrix Love

This is a Matrix code rain animation in Love2d engine

## Requirements

- Love2D engine
- Lua (basically installed with Love2D engine)

## How to run

- Run `love .` on project dir
- Or run `zip -r matrix.love *` on the root
- And double click the `matrix.love` file
